1. What is Machine Learning?

Machine Learning (ML) enables systems to automatically learn and make decisions from data without being explicitly programmed. It's a subset of Artificial Intelligence (AI), which aims to simulate human intelligence.

üß≠ Workflow:

  1. Define the problem (classification, regression, clustering).
  2. Gather and preprocess data (clean, handle missing values).
  3. Feature selection/engineering (normalize, encode, reduce).
  4. Choose the algorithm and train the model.
  5. Evaluate performance using metrics.
  6. Deploy and monitor for real-world adaptation.

4. Deep Learning

Deep Learning uses multi-layered neural networks for automatic feature extraction and representation.

  • ANN ‚Äî Fully connected networks for structured data.
  • CNN ‚Äî Used for image data processing.
  • RNN/LSTM/GRU ‚Äî Used for sequential data like text and audio.
  • Transformers ‚Äî State-of-the-art architecture for NLP and beyond.
