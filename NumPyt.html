np

Advanced NumPy Cheat Sheet

A fully detailed, structured, and visually enhanced reference for learning NumPy fundamentals, advanced array operations, broadcasting rules, linear algebra, random operations, indexing, slicing, reshaping, and ML-specific tricks.

1. Array Creation

Basic

np.array([1,2,3])               # create array
np.array([[1,2],[3,4]])         # 2D array
np.asarray(list_obj)            # convert list → ndarray

Special Arrays

np.zeros((3,3))                 # matrix of 0s
np.ones((2,4))                  # matrix of 1s
np.full((2,2), 7)               # matrix filled with 7
np.eye(4)                       # identity matrix

Sequences

np.arange(0, 20, 2)             # even sequence
np.linspace(0, 1, 10)           # 10 evenly spaced points
np.logspace(1, 3, 5)            # log scale spacing

2. Random Numbers

Basic

np.random.rand(3,3)             # uniform dist [0,1)
np.random.randn(3,3)            # normal dist (0,1)
np.random.randint(1, 50, (3,3)) # random ints

Advanced Random

np.random.seed(42)              # reproducibility
np.random.uniform(-1,1,10)      # uniform range
np.random.normal(5, 2, 100)     # mean=5 std=2

3. Linear Algebra

Matrix Operations

A @ B                          # matrix mul
np.dot(A,B)                      # dot product
np.transpose(A)                  # transpose

Inverses & Decomposition

np.linalg.inv(A)                # inverse
np.linalg.pinv(A)               # pseudo-inverse
np.linalg.det(A)                # determinant
np.linalg.eig(A)                # eigenvalues

4. Statistical Functions

Basics

np.mean(x)
np.median(x)
np.std(x)
np.var(x)

Aggregations

np.min(x), np.max(x)
np.percentile(x, 90)
x.sum(), x.prod()

5. Reshaping & Dimensions

Shape Changing

a.reshape(3,4)
a.flatten()
a.ravel()

Dim Helpers

np.expand_dims(a, axis=0)
a.squeeze()
a.swapaxes(0,1)

6. Indexing & Slicing

Basics

a[1:4]
a[::2]
a[-1]

Boolean

a[a > 10]
np.where(a > 5)
np.where(a % 2 == 0, "even", "odd")

7. Broadcasting Rules (Advanced)

# shapes must be compatible:
# trailing dimensions must match or be 1

(3,1)  + (1,4)   → (3,4)
(5,1,4) + (1,3,1) → (5,3,4)

# ML example: normalize batch
X_norm = (X - X.mean(0)) / X.std(0)

8. NumPy for Machine Learning

Feature Scaling

X_norm = (X - X.mean(axis=0)) / X.std(axis=0)

Adding Bias

X_b = np.c_[np.ones((len(X),1)), X]

Linear Regression (Normal Equation)

theta = np.linalg.pinv(X.T @ X) @ X.T @ y
