Umer's Detailed NumPy Cheatsheet

1. Array Creation

  • np.array(): Create a NumPy array 
    a = np.array([1,2,3])  # [1 2 3]
  • np.zeros(shape): Array filled with zeros 
    np.zeros((2,3))  # [[0. 0. 0.] [0. 0. 0.]]
  • np.ones(shape): Array filled with ones 
    np.ones((2,3))  # [[1. 1. 1.] [1. 1. 1.]]
  • np.arange(start,end,step): Sequence array 
    np.arange(0,5)  # [0 1 2 3 4]
  • np.linspace(start,end,num): Evenly spaced array 
    np.linspace(0,1,5)  # [0. 0.25 0.5 0.75 1.]
  • np.random.rand(dims): Random floats 0-1 
    np.random.rand(2,3)
  • np.random.randn(dims): Random normal distribution 
    np.random.randn(2,3)
  • np.random.randint(low,high,size): Random integers 
    np.random.randint(1,10,(2,2))

2. Array Properties

  • shape: a.shape â†’ dimensions tuple
  • dtype: a.dtype â†’ element type
  • size: a.size â†’ total elements
  • ndim: a.ndim â†’ number of dimensions

3. Basic Operations

  • Addition, multiplication, min, max, mean, std, sum
    • a = np.array([1,2,3])
a + 5   # [6 7 8]
a * 2   # [2 4 6]
np.min(a)  # 1
np.max(a)  # 3
a.mean()   # 2.0
np.std(a)  # 0.816

4. Indexing & Slicing

  • 1D: a[1] â†’ second element
  • Slice: a[1:4] â†’ elements 1-3
  • 2D: a[1,2] â†’ row 1, col 2
  • Row: a[1,:]
  • Column: a[:,2]
  • Boolean: a[a>3] â†’ select values where condition is True

5. Reshaping & Dimensions

  • reshape(): 
    a.reshape(2,3)
  • resize(): in-place reshape 
    a.resize(2,3)
  • expand_dims(): add dimension 
    np.expand_dims([1,2,3], axis=0)  # [[1 2 3]]
  • squeeze(): remove singleton dimensions 
    np.squeeze([[[1,2,3]]])  # [1 2 3]

6. Stacking & Concatenation

  • vstack(): stack vertically 
    np.vstack((a,b))
  • hstack(): stack horizontally 
    np.hstack((a,b))
  • concatenate(): merge arrays along axis 
    np.concatenate((a,b), axis=0/1)

7. Linear Algebra

  • Dot Product: np.dot(a,b)
  • Matrix multiplication: np.matmul(a,b)
  • Inverse: np.linalg.inv(a)
  • Index of largest element: np.argmax(a)
  • Index of smallest element: np.argmin(a)

8. Sorting & Clipping

  • Sort along axis: np.sort(a, axis=0/1)
  • Clip values: np.clip([1,5,10],2,8) # [2 5 8]

9. Random & Padding

  • Pad array: 
    np.pad([1,2,3,4], (2,1), constant_values=0)  # [0 0 1 2 3 4 0]
  • Random numbers: np.random.rand(), np.random.randn(), np.random.randint()

10. Vectorization & Views

  • Vectorization: operate on arrays directly 
    a = np.array([1,2,3]); b = a + 5  # [6 7 8]
  • copy vs view: 
    b = a.copy()  # independent
c = a.view()  # changes affect original
  • Slicing returns view: modifying slice affects original
