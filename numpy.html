NumPy Cheatsheet — Beginner

Compact reference: arrays, creation, math, stats, indexing, random, and matrix ops.

1. Import

import numpy as np

2. Creating arrays

  • np.array() — create array from list: np.array([1,2,3])
  • np.zeros(shape) — zeros: np.zeros((3,3))
  • np.ones(shape) — ones: np.ones((2,4))
  • np.full(shape, value) — fill: np.full((2,3), 5)
  • np.arange(start, stop, step) — range: np.arange(0,10,2)
  • np.linspace(start, stop, num) — evenly spaced: np.linspace(0,1,5)

3. Array info & shape

  • arr.shape — shape tuple
  • arr.ndim — number of dimensions
  • arr.size — total elements

4. Reshaping

a = np.arange(6)
b = a.reshape(2,3)
flat = a.flatten()

5. Math & Statistics

  • np.add, np.subtract, np.multiply, np.divide — elementwise ops
  • np.power(a,2), np.sqrt(a)
  • np.mean(a), np.median(a), np.sum(a), np.min(a), np.max(a), np.std(a)

6. Indexing & slicing

# 1D
x = a[0]
y = a[-1]
slice = a[1:4]

# 2D
val = b[0,1]
col = b[:,0]
row = b[1,:]

7. Random

  • np.random.randint(low,high,size) — ints
  • np.random.randn(rows,cols) — normal samples
  • np.random.random(shape) — floats in [0,1)

8. Matrix ops

  • np.dot(a,b) or a @ b
  • a.T — transpose
  • np.linalg.inv(mat) — inverse

9. Example

import numpy as np

# create
A = np.arange(6).reshape(2,3)
B = np.ones((2,3))

# math & stats
C = A + B
mean_all = np.mean(C)

print(A)
print(C)
print("mean:", mean_all)
